# eZags
## Vital Records Management Software
## How to run
~~~
cd OneByte
composer install
php artisan key:generate
nano .env
mysql your_database < db.sql
php artisan serve
~~~
In browser open: http://localhost:8080
### Requirements
* **SERVER**: Apache 2 or NGINX.
* **RAM**: 1 GB or higher.
* **PHP**: 7.3 or higher.
* **MySQL or MariaDB**
* **Composer**
